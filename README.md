# PC-1500_ROM_Modules

ROM modules for Sharp PC-1500 which have been dumped. Typically each ROMs folder will have a notes.txt file which specific notations for that ROM.

File extesnions:
.ASC  -- Text file (ASCII) of BASIC program with CR line endings
.BAS  -- Tokenized BASIC file with CE-158 header
.BIN  -- Binary dump of ROM, may be read directly from ROM or dumped over serial.
.FRAG -- Fragment file for LHTOOLS
.RSV  -- Reserve memory dump from cart with CE-158 header
.SBS  -- Text file (ASCII) of BASIC program with CRLF line endings used with SBASIC VSCOCDE extension
.TXT  -- Text file (ASCII) of BASIC program with random line endings because GIT likes to change line endings
.wav  -- Audio of program saved to cassette
